- 开放 API

### 2.3 技术优势

```python
技术栈对比:

传统方案:
└─ 手动标签 + 文件名搜索
   问题: 费时费力，准确率低

Google/Apple:
└─ 通用 CLIP/ViT 模型
   问题: 中文 OCR 弱 (20-40%)

我们的方案:
├─ Qwen2.5-VL (中文 OCR 95%+)
├─ SigLIP (视觉编码)
├─ BGE-M3 (文本编码)
└─ 混合检索引擎
   优势: 准确率 95%+, 全方位覆盖
```
